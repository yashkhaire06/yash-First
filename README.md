# yash-First
A command-line game built in Python where the computer picks a random number and the player tries to guess it.

## What it does

- Computer secretly picks a number between 1 and 100
- Player guesses and gets "too high" or "too low" feedback
- Tracks number of attempts and gives a performance rating
- Handles invalid input without crashing
- Asks if you want to play again after each round

## How to run it

Make sure you have Python installed, then run:

```
python game.py
```

No extra libraries needed. Just Python.
